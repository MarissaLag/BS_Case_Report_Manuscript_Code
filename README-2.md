###Code for Baynes Sound Case Report manuscript###

#Code organized into three sections:

#1. Marine heatwave (MHW) analyses - Includes modified data (excel sheet) used to calculate MHW events (frequency, duration, intensity) and modelling methods used to analyze MHW characteristics over time (1969-2018) (R script).

#2. Shellfish production statistics - Includes modified data (excel sheet) containing only shellfish sales in British Columbia and modelling methods used to characterize shellfish growth and plataeus overtime (1986-2023) (R script).

#3. Baynes Sound map making code (R script).

#Data sourced for each section:

#1. http://www.pac.dfo-mpo.gc.ca/science/oceans/data-donnees/lightstations-phares/index-eng.html (temperature data) and https://psl.noaa.gov/data/correlation/oni.csv (ENSO classifications). 

#2. https://www.dfo-mpo.gc.ca/stats/aqua/aqua-prod-eng.htm (aquaculture data)
